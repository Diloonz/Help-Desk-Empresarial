# Help-Desk-Empresarial
Um aplicativo de demonstração de help desk empresarial no ramo ambiental.

1- Aplicativo contém funções de controle dos equipamentos da empresa para maior controle de logistica.
2- Contém armazenamentos de resultados de amostras do laboratório.
3- Dados de todos os clientes da empresa, podendo localizar por buscas em tempo real.
