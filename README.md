# Help-Desk-Empresarial

## Visão Geral
Este é um aplicativo de demonstração de Help Desk empresarial desenvolvido no Microsoft Power Apps, focado em soluções para o ramo ambiental. Ele foi projetado para otimizar o controle logístico de equipamentos, o gerenciamento de dados de amostras de laboratório e o cadastro e busca de informações de clientes.

## Problema Resolvido / Objetivos
O projeto foi criado para solucionar desafios comuns em empresas do ramo ambiental, como:
* Falta de um sistema centralizado para controle de equipamentos e sua logística.
* Dificuldade no armazenamento e acesso rápido aos resultados de amostras de laboratório.
* Gerenciamento ineficiente de dados de clientes, incluindo funcionalidades de busca.

## Funcionalidades Principais
* **Controle Logístico de Equipamentos:** Gerenciamento de entrada/saída e localização de equipamentos.
* **Armazenamento de Resultados de Amostras:** Registro e consulta de dados de amostras de laboratório.
* **Gerenciamento de Clientes:** Cadastro completo e sistema de busca em tempo real por clientes.

## Tecnologias Utilizadas
* **Microsoft Power Apps:** Desenvolvimento da interface e lógica de negócios low-code.
* **SharePoint Lists:** Utilizado para armazenamento e recuperação de dados.

## Como Funciona

O aplicativo permite que diferentes usuários gerenciem o ciclo de vida dos serviços:

* **Gestão de Equipamentos:** O funcionário em campo (coletor) pode atualizar o status de um equipamento em tempo real (ex: Disponível, Em Manutenção, Com Defeito). Isso dá à equipe de logística um controle preciso para otimizar a alocação de coletores.

* **Gestão de Amostras:** O app possui uma funcionalidade para salvar e consultar os dados e resultados das amostras analisadas pelo laboratório.

* **Gestão de Clientes:** Centraliza o cadastro de todos os clientes em um único local, permitindo pesquisar e visualizar seus dados rapidamente.
